Built In Pipes

Built is Pipes are those which are provided by the angular by default.

  • lowercase
  • uppercase
  • titlecase
  • slice
  • json
  • decimal
  • percent
  • currency
  • data
  • async

In the number pipe by default values are "minIntegerDigit = 1, minFractionDigit = 0, maxFractionDigit = 3"

syntax: minIntegerDigit . minFractionDigit - maxFractionDigit

{{12.96862493 | number}}

{{10.96862493 | number:'1.2-5'}}

{{10.96862493 | number:'3.0-3'}}

{{ 3.5 | percent:'1.2-5' }}

{{879.877 | currency:'INR':true:'1.2-4'}}
