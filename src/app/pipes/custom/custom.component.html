Custom Pipes

Custom Pipes are the pipes that are created by the user to fullfill the specific type of need. It is defined under the Pipe decorator which we can impor from the angular core library. The Pipe allow you define the pipe name that you will use within template expression.

for example: Below I have create a to add a prefix and postfix in a value.

{{'Abhi' | prefixPostfix:{prefix:'custom-', postfix:'-pipe'} }}
