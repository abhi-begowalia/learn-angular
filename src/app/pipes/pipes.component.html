Pipes in Angular

Angular Pipes are used to transform data to desired output. Pipes take the data as a input and transform data to a different output. Using pipe oprator(|), we acan apply the Pipes's features to any of the property in our angular project.

Pipes (|) in Angular are used to transform data before displaying it in the browser. Angular provides a lot of built in pipes to translate the data before displaying in into the browser and we know, Angular lets us extend its features, we can even create custom pipes in Angular.

There are basically two types of Pipes in Angular

  • Built In Pipes
    • Parameterized
    • chaining
  • Custom Pipes
