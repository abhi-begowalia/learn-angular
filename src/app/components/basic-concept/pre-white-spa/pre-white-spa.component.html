Preserve White Space: Preserve White Space is meta element in the Component decortor which is used to apply the default space which is given in the html file. It has a two values which is true and false. By default it has a false value which means that don't take the white space between the elements. It will be define by the preserveWhitespaces property. For example you can see the bottom we have added the space between the span elements.
