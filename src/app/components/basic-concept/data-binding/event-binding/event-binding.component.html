Event Binding

When a user interacts with an application in the form of a keyboard movement, a mouse click, or a mouseover, it generates an event. These events need to be handled to perform some kind of action. This is where event binding comes into the picture. Event Binding is also one way binding. But in this bindind Data will be flow in View to component.

(click)="onClick()"

In this case, onClick() method of the component class is called when the click event occurs.
