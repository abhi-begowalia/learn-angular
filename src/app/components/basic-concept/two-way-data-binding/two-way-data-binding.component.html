Two Way Data Binding

It is most used mechanism of data binding. Basically two way data binding is used mostly with input field or any form element where user type or provide any value or change any control value in one side, and on other side, the same automatically updated in to the controller variable and vice versa.

In simple words, two way data binding is a combination of both Property Binding and Event Binding. It is used on that situation when we want to use the property binding and event binding on same element.
