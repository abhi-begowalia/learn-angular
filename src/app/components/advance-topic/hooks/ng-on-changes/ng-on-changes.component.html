ngOnChanges: The ngOnChanges hook is used to detect the input changes. For example if you have a variable in the parent component and you are using its values in the child component. variable will automatically detect the changes and update the value. But If you want to perform a any specific action on value changes then you can use the ngOnChanges hook. Whenever anything will be changed it will automatically call. It will be called after the Constructor.

Use Cases: Use it when you need to respond to changes in input properties and perform logic based on those changes.
