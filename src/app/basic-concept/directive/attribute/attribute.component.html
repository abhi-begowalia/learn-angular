Attribute Directive in Angular

Thee attributes directive changes the apperance or behavior of a DOM elements. These directive look live regular HTML attributes in template. The ngModel directive which is used for two-day is an example of an attributes directive. Some of other attribute directive are listed below:

  • Style: used for dynamic style can be set by using NgStyle.
  • Class: used for adding and removing css classes dynamically.
